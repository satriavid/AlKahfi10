# Hafal Al-Kahfi 10

 Aplikasi web untuk menghafal 10 ayat pertama Surah Al-Kahfi. Dibangun dengan React, TypeScript, Tailwind CSS, dan Vite.

## Fitur

- Pembelajaran ayat per ayat dengan terjemahan bahasa Indonesia
- Audio pengajian dari Mishary Rashid Alafasy dan Abdurrahman As-Sudais
- Tafsir dan tadabbur untuk setiap ayat
- Progress menghafal dengan indikator selesai
- Pengaturan kecepatan putar audio
- Mode loop untuk pengulangan

## Teknologi

- React 19
- TypeScript
- Tailwind CSS
- Vite
- Lucide React (icons)

## Menjalankan Aplikasi

1. Install dependencies:
   ```bash
   npm install
   ```

2. Jalankan aplikasi:
   ```bash
   npm run dev
   ```

3. Buka http://localhost:3000 di browser

## Build untuk Produksi

```bash
npm run build
```
